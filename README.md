# 📊 Regional Sales Analysis 2025

## 📌 Project Overview

The **Regional Sales Analysis 2025** project analyzes retail sales data to identify revenue bottlenecks, customer behavior, product performance, and sales trends across different regions. The project demonstrates how SQL and Microsoft Excel can be used together to extract meaningful business insights and support data-driven decision-making.

This project was completed as part of a **Business Analyst Practical** using **MySQL** for data analysis and **Microsoft Excel** for dashboard visualization.

---

## 🎯 Objectives

- Analyze monthly sales performance.
- Identify regions with the highest revenue.
- Calculate cancellation and return percentages.
- Determine the top-performing sales agents.
- Measure category-wise sales contribution.
- Identify products causing maximum revenue loss.
- Detect customers with frequent product returns.
- Create an interactive Excel dashboard for business reporting.

## 🔍 Key Insights

- Monthly sales trends help identify seasonal demand.
- Revenue varies across different regions.
- Electronics and Home Appliances contribute significantly to total sales.
- Cancelled and returned orders reduce overall profitability.
- Top-performing sales agents consistently generate higher completed sales.
- Frequent customer returns highlight opportunities for quality improvement.

---

## 💡 Business Recommendations

- Improve inventory planning to reduce cancellations.
- Review products with high return rates.
- Reward top-performing sales agents.
- Strengthen marketing efforts in lower-performing regions.
- Monitor customer return patterns regularly.
- Use monthly sales trends for demand forecasting.
